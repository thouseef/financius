# Project setup
- Create a file `private.properties` in root directory.
```
CRASHLYTICS=your_id_goes_here
GOOGLE_WEB_CLIENT_ID="your_id_goes_here"
GOOGLE_WEB_CLIENT_ID_DEBUG="your_id_goes_here"
```
- Add google-services.json from Firebase.
